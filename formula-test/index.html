덧셈·뺄셈 공식

sin(α + β) =sinα cosβ + cosα sinβ
sin(α − β) =sinα cosβ − cosα sinβ
cos(α + β) =cosα cosβ − sinα sinβ
cos(α − β) =cosα cosβ + sinα sinβ

배각·반각

sin2θ =2 sinθ cosθ
cos2θ =cos²θ − sin²θ = 2cos²θ − 1 = 1 − 2sin²θ
sin²(θ/2) =(1 − cosθ) / 2
cos²(θ/2) =(1 + cosθ) / 2

합↔곱, 곱↔합

2 sinα cosβ =sin(α+β) + sin(α−β)
2 cosα cosβ =cos(α+β) + cos(α−β)
sinα + sinβ =2 sin((α+β)/2) cos((α−β)/2)

선형대수 핵심

cos θ (내적) =(u · v) / (‖u‖ · ‖v‖)
직교 조건: u · v =0
proj_v(u) =(u·v)/(v·v) · v
투영 길이 =‖u‖ cosθ = (u·v)/‖v‖
2D det([u v]) =u1 v2 − u2 v1 = ‖u‖·‖v‖·sin θ
3D ‖u × v‖ =‖u‖·‖v‖·sin θ

미적분 핵심

d/dx[sin x] =cos x
d/dx[cos x] =−sin x
∫ sin x dx =−cos x + C
∫ cos x dx =sin x + C

작은각 근사
